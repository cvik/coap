# DTLS 1.2 Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add DTLS 1.2 PSK security to the CoAP server and client, pure Zig, zero C deps.

**Architecture:** Layered modules under `src/dtls/` — crypto primitives (Ccm, Prf), wire format (Record, Cookie), session management (Session), and handshake state machine (Handshake). Server.zig and Client.zig integrate DTLS transparently when PSK credentials are provided in config.

**Tech Stack:** Zig, `std.crypto.core.aes.Aes128`, `std.crypto.auth.hmac.sha2.HmacSha256`, `std.crypto.hash.sha2.Sha256`, `std.crypto.utils.secureZero` (verify exact path in your Zig version — may be `std.crypto.secureZero`).

**Threading:** Each server thread gets its own `SessionTable` (same pattern as Exchange and RateLimiter). With `SO_REUSEPORT`, the kernel hashes by source IP:port, so a given peer consistently reaches the same thread. DTLS sessions are per-thread, no shared state.

**IPv6:** The current codebase is IPv4-only (`Io.zig:80` rejects non-INET). DTLS follows the same constraint. IPv6 DTLS is future work.

**Spec:** `docs/superpowers/specs/2026-03-12-dtls-design.md`

### Task 2: AES-128-CCM-8 (Ccm.zig)

**Files:**
- Create: `src/dtls/Ccm.zig`

**Reference:** RFC 3610 (CCM mode), RFC 6655 (AES-CCM for TLS)

CCM = CTR encryption + CBC-MAC authentication. The construction:
1. Generate CBC-MAC tag over: flags/lengths block (B0), additional data, plaintext
2. Encrypt plaintext with AES-CTR (counter starts at 1)
3. Encrypt the tag with AES-CTR counter 0

For CCM-8: tag is truncated to 8 bytes. Nonce is 12 bytes, leaving 3 bytes for counter
(L=2 in CCM terms, so max message length = 2^16 - 1 = 65535 bytes, sufficient for DTLS).

- [ ] **Step 3: Implement AES-128-CCM-8**

Implementation in `src/dtls/Ccm.zig`. The CCM construction with L=2 (2-byte length field),
nonce length = 12, tag length = 8:

**NOTE:** The AES API in Zig's stdlib uses `Aes128.initEnc()` which returns an
`AesEncryptCtx(Aes128)`, not `Aes128` itself. The `encrypt` method signature is
`encrypt(dst: *[16]u8, src: *const [16]u8)` — destination first. Verify exact
API against your Zig version. Similarly, `std.crypto.utils.timingSafeEql` and
`std.crypto.utils.secureZero` paths may differ — check `std.crypto` namespace.

```zig
const std = @import("std");
const Aes128 = std.crypto.core.aes.Aes128;
const AesEncCtx = Aes128.EncryptCtx; // verify exact type name

const tag_len = 8;
const nonce_len = 12;
// CCM L = 15 - nonce_len = 3 (3-byte counter field, max msg 2^24-1 bytes)

/// Encrypt plaintext with AES-128-CCM-8.
/// `out` must be `plaintext.len + 8` bytes (ciphertext + 8-byte tag).
pub fn encrypt(
    plaintext: []const u8,
    ad: []const u8,
    nonce: [nonce_len]u8,
    key: [16]u8,
    out: []u8,
) void {
    std.debug.assert(out.len == plaintext.len + tag_len);
    const aes = Aes128.initEnc(key);

    // 1. Compute CBC-MAC tag
    var tag = computeCbcMac(aes, nonce, ad, plaintext);

    // 2. Encrypt tag with counter 0
    const ctr0_block = formatCounter(nonce, 0);
    var ctr0_enc: [16]u8 = undefined;
    aes.encrypt(&ctr0_enc, &ctr0_block); // dst, src
    for (0..tag_len) |i| {
        tag[i] ^= ctr0_enc[i];
    }

    // 3. Encrypt plaintext with CTR starting at counter 1
    ctrEncrypt(aes, nonce, plaintext, out[0..plaintext.len]);

    // 4. Append truncated tag
    @memcpy(out[plaintext.len..][0..tag_len], tag[0..tag_len]);
}

/// Decrypt ciphertext with AES-128-CCM-8.
/// `ciphertext_and_tag` is `N + 8` bytes. `out` must be `N` bytes.
pub fn decrypt(
    ciphertext_and_tag: []const u8,
    ad: []const u8,
    nonce: [nonce_len]u8,
    key: [16]u8,
    out: []u8,
) error{AuthenticationFailed}!void {
    if (ciphertext_and_tag.len < tag_len) return error.AuthenticationFailed;
    const ct_len = ciphertext_and_tag.len - tag_len;
    std.debug.assert(out.len == ct_len);
    const aes = Aes128.initEnc(key);

    // 1. Decrypt ciphertext with CTR starting at counter 1
    ctrEncrypt(aes, nonce, ciphertext_and_tag[0..ct_len], out);

    // 2. Compute CBC-MAC over decrypted plaintext
    var expected_tag = computeCbcMac(aes, nonce, ad, out);

    // 3. Decrypt expected tag with counter 0
    const ctr0_block = formatCounter(nonce, 0);
    var ctr0_enc: [16]u8 = undefined;
    aes.encrypt(&ctr0_enc, &ctr0_block); // dst, src
    for (0..tag_len) |i| {
        expected_tag[i] ^= ctr0_enc[i];
    }

    // 4. Verify tag (constant-time compare)
    const received_tag = ciphertext_and_tag[ct_len..][0..tag_len];
    // NOTE: verify exact path — may be std.crypto.timing_safe.eql or similar
    if (!std.crypto.utils.timingSafeEql([tag_len]u8, received_tag.*, expected_tag[0..tag_len].*)) {
        std.crypto.utils.secureZero(u8, out);
        return error.AuthenticationFailed;
    }
}

/// Format a CCM counter block: flags(1) || nonce(12) || counter(3)
fn formatCounter(nonce: [nonce_len]u8, counter: u24) [16]u8 {
    var block: [16]u8 = undefined;
    block[0] = 2; // flags = L-1 = 2
    @memcpy(block[1..][0..nonce_len], &nonce);
    block[13] = @intCast((counter >> 16) & 0xFF);
    block[14] = @intCast((counter >> 8) & 0xFF);
    block[15] = @intCast(counter & 0xFF);
    return block;
}

/// Compute CBC-MAC tag for CCM.
/// NOTE: aes.encrypt may not support in-place aliasing (dst == src).
/// If your Zig version prohibits aliasing, use a temp buffer for CBC-MAC steps.
fn computeCbcMac(aes: AesEncCtx, nonce: [nonce_len]u8, ad: []const u8, plaintext: []const u8) [16]u8 {
    var b0: [16]u8 = undefined;
    const adata_flag: u8 = if (ad.len > 0) 0x40 else 0;
    b0[0] = adata_flag | (3 << 3) | 2; // adata | M'=3 | L'=2
    @memcpy(b0[1..][0..nonce_len], &nonce);
    const mlen: u24 = @intCast(plaintext.len);
    b0[13] = @intCast((mlen >> 16) & 0xFF);
    b0[14] = @intCast((mlen >> 8) & 0xFF);
    b0[15] = @intCast(mlen & 0xFF);

    // Start CBC-MAC with B0
    var mac: [16]u8 = undefined;
    aes.encrypt(&mac, &b0); // dst, src

    // Encode additional data length + data
    if (ad.len > 0) {
        var ad_block: [16]u8 = .{0} ** 16;
        std.debug.assert(ad.len < 65280);
        ad_block[0] = @intCast((ad.len >> 8) & 0xFF);
        ad_block[1] = @intCast(ad.len & 0xFF);
        const first_chunk = @min(ad.len, 14);
        @memcpy(ad_block[2..][0..first_chunk], ad[0..first_chunk]);
        xorBlock(&mac, &ad_block);
        var tmp: [16]u8 = undefined;
        aes.encrypt(&tmp, &mac); // use tmp to avoid aliasing
        mac = tmp;

        var ad_offset: usize = first_chunk;
        while (ad_offset < ad.len) {
            @memset(&ad_block, 0);
            const chunk = @min(ad.len - ad_offset, 16);
            @memcpy(ad_block[0..chunk], ad[ad_offset..][0..chunk]);
            xorBlock(&mac, &ad_block);
            aes.encrypt(&tmp, &mac);
            mac = tmp;
            ad_offset += chunk;
        }
    }

    var pt_offset: usize = 0;
    while (pt_offset < plaintext.len) {
        var pt_block: [16]u8 = .{0} ** 16;
        const chunk = @min(plaintext.len - pt_offset, 16);
        @memcpy(pt_block[0..chunk], plaintext[pt_offset..][0..chunk]);
        xorBlock(&mac, &pt_block);
        var tmp: [16]u8 = undefined;
        aes.encrypt(&tmp, &mac);
        mac = tmp;
        pt_offset += chunk;
    }

    return mac;
}

/// CTR-mode encrypt/decrypt (symmetric).
fn ctrEncrypt(aes: AesEncCtx, nonce: [nonce_len]u8, input: []const u8, output: []u8) void {
    var counter: u24 = 1;
    var offset: usize = 0;
    while (offset < input.len) {
        const ctr_block = formatCounter(nonce, counter);
        var keystream: [16]u8 = undefined;
        aes.encrypt(&keystream, &ctr_block); // dst, src
        const chunk = @min(input.len - offset, 16);
        for (0..chunk) |i| {
            output[offset + i] = input[offset + i] ^ keystream[i];
        }
        offset += chunk;
        counter += 1;
    }
}

fn xorBlock(dst: *[16]u8, src: *const [16]u8) void {
    for (dst, src) |*d, s| d.* ^= s;
}
```

```zig
const std = @import("std");
const HmacSha256 = std.crypto.auth.hmac.sha2.HmacSha256;

/// TLS 1.2 PRF using HMAC-SHA-256 (RFC 5246 §5).
/// PRF(secret, label, seed) = P_SHA256(secret, label || seed)
/// P_hash(secret, seed) = HMAC(secret, A(1)||seed) || HMAC(secret, A(2)||seed) || ...
/// A(0) = seed, A(i) = HMAC(secret, A(i-1))
pub fn prf(
    secret: []const u8,
    label: []const u8,
    seed: []const u8,
    out: []u8,
) void {
    // A(0) = label || seed
    // We compute HMAC incrementally to avoid concatenation allocation.
    // A(1) = HMAC(secret, A(0)) = HMAC(secret, label || seed)

    var a: [HmacSha256.mac_length]u8 = undefined;

    // Compute A(1) = HMAC(secret, label || seed)
    var h = HmacSha256.init(secret);
    h.update(label);
    h.update(seed);
    h.final(&a);

    var offset: usize = 0;
    while (offset < out.len) {
        // P(i) = HMAC(secret, A(i) || label || seed)
        var p = HmacSha256.init(secret);
        p.update(&a);
        p.update(label);
        p.update(seed);
        var block: [HmacSha256.mac_length]u8 = undefined;
        p.final(&block);

        const remaining = out.len - offset;
        const copy_len = @min(remaining, HmacSha256.mac_length);
        @memcpy(out[offset..][0..copy_len], block[0..copy_len]);
        offset += copy_len;

        // A(i+1) = HMAC(secret, A(i))
        var a_next = HmacSha256.init(secret);
        a_next.update(&a);
        a_next.final(&a);
    }
}
```

/// Generate a stateless DTLS cookie.
/// Cookie = HMAC-SHA256(server_secret, client_addr || client_random)
pub fn generate(
    server_secret: [32]u8,
    client_addr: std.net.Address,
    client_random: [32]u8,
) [32]u8 {
    var h = HmacSha256.init(&server_secret);
    const addr_bytes: [16]u8 = @bitCast(client_addr.any);
    h.update(&addr_bytes);
    h.update(&client_random);
    var cookie: [32]u8 = undefined;
    h.final(&cookie);
    return cookie;
}

/// Verify a DTLS cookie against the server secret and client parameters.
/// Tries current secret first, then previous (for rotation window).
pub fn verify(
    server_secret: [32]u8,
    client_addr: std.net.Address,
    client_random: [32]u8,
    cookie: []const u8,
) bool {
    return verifySingle(server_secret, client_addr, client_random, cookie);
}

/// Verify with rotation: try current secret, then previous.
pub fn verifyWithRotation(
    current_secret: [32]u8,
    previous_secret: [32]u8,
    client_addr: std.net.Address,
    client_random: [32]u8,
    cookie: []const u8,
) bool {
    if (verifySingle(current_secret, client_addr, client_random, cookie)) return true;
    return verifySingle(previous_secret, client_addr, client_random, cookie);
}

fn verifySingle(
    server_secret: [32]u8,
    client_addr: std.net.Address,
    client_random: [32]u8,
    cookie: []const u8,
) bool {
    if (cookie.len != 32) return false;
    const expected = generate(server_secret, client_addr, client_random);
    return std.crypto.utils.timingSafeEql([32]u8, cookie[0..32].*, expected);
}

const std = @import("std");
const Ccm = @import("Ccm.zig");
const types = @import("types.zig");

pub const Record = struct {
    content_type: types.ContentType,
    epoch: u16,
    sequence_number: u48,
    payload: []const u8,
};

/// Encode a plaintext DTLS record (epoch 0).
/// Writes header + payload into `out`. Advances `sequence_number`.
/// Returns the encoded slice.
pub fn encodePlaintext(
    content_type: types.ContentType,
    payload: []const u8,
    sequence_number: *u48,
    out: []u8,
) []const u8 {
    const seq = sequence_number.*;
    const total = types.record_header_len + payload.len;
    std.debug.assert(out.len >= total);

    writeHeader(out, content_type, 0, seq, @intCast(payload.len));
    @memcpy(out[types.record_header_len..][0..payload.len], payload);

    sequence_number.* += 1;
    return out[0..total];
}

/// Encode an encrypted DTLS record (epoch >= 1).
/// Wire format: header(13) || explicit_nonce(8) || ciphertext || tag(8)
/// Returns the encoded slice.
pub fn encodeEncrypted(
    content_type: types.ContentType,
    plaintext: []const u8,
    write_key: [16]u8,
    write_iv: [4]u8,
    epoch: u16,
    sequence_number: *u48,
    out: []u8,
) []const u8 {
    const seq = sequence_number.*;
    const ct_len = plaintext.len + types.ccm8_tag_len;
    const total = types.record_header_len + types.explicit_nonce_len + ct_len;
    std.debug.assert(out.len >= total);

    // Record header — length field includes explicit_nonce + ciphertext + tag
    writeHeader(out, content_type, epoch, seq, @intCast(types.explicit_nonce_len + ct_len));

    // Explicit nonce: epoch(2) || sequence(6)
    const explicit_nonce = buildExplicitNonce(epoch, seq);
    @memcpy(out[types.record_header_len..][0..types.explicit_nonce_len], &explicit_nonce);

    // Nonce = implicit_iv(4) || explicit_nonce(8)
    const nonce = buildNonce(write_iv, explicit_nonce);

    // Additional data = record header with length = plaintext length
    var ad: [types.record_header_len]u8 = undefined;
    writeHeader(&ad, content_type, epoch, seq, @intCast(plaintext.len));

    // Encrypt
    const ct_start = types.record_header_len + types.explicit_nonce_len;
    Ccm.encrypt(plaintext, &ad, nonce, write_key, out[ct_start..][0..ct_len]);

    sequence_number.* += 1;
    return out[0..total];
}

/// Decode a plaintext DTLS record (epoch 0).
pub fn decodePlaintext(buf: []const u8) ?Record {
    if (buf.len < types.record_header_len) return null;
    const hdr = parseHeader(buf) orelse return null;
    if (buf.len < types.record_header_len + hdr.length) return null;
    return .{
        .content_type = hdr.content_type,
        .epoch = hdr.epoch,
        .sequence_number = hdr.sequence_number,
        .payload = buf[types.record_header_len..][0..hdr.length],
    };
}

/// Decode and decrypt a DTLS record (epoch >= 1).
/// Performs anti-replay check. Returns null on replay, auth failure, or malformed record.
pub fn decodeEncrypted(
    buf: []const u8,
    read_key: [16]u8,
    read_iv: [4]u8,
    replay_window: *u64,
    max_seq: *u48,
    plaintext_buf: []u8,
) ?Record {
    if (buf.len < types.record_header_len) return null;
    const hdr = parseHeader(buf) orelse return null;

    // Must have at least explicit_nonce + tag
    if (hdr.length < types.explicit_nonce_len + types.ccm8_tag_len) return null;
    if (buf.len < types.record_header_len + hdr.length) return null;

    const payload_start = types.record_header_len;
    const explicit_nonce = buf[payload_start..][0..types.explicit_nonce_len];
    const ct_and_tag = buf[payload_start + types.explicit_nonce_len ..][0 .. hdr.length - types.explicit_nonce_len];
    const pt_len = ct_and_tag.len - types.ccm8_tag_len;

    // Anti-replay check
    if (!replayCheck(hdr.sequence_number, replay_window, max_seq)) return null;

    // Build nonce and AD
    const nonce = buildNonce(read_iv, explicit_nonce[0..types.explicit_nonce_len].*);
    var ad: [types.record_header_len]u8 = undefined;
    writeHeader(&ad, hdr.content_type, hdr.epoch, hdr.sequence_number, @intCast(pt_len));

    // Decrypt
    Ccm.decrypt(ct_and_tag, &ad, nonce, read_key, plaintext_buf[0..pt_len]) catch {
        // Auth failed — undo replay window update
        replayUndo(hdr.sequence_number, replay_window, max_seq);
        return null;
    };

    return .{
        .content_type = hdr.content_type,
        .epoch = hdr.epoch,
        .sequence_number = hdr.sequence_number,
        .payload = plaintext_buf[0..pt_len],
    };
}

// --- Internal helpers ---

const Header = struct {
    content_type: types.ContentType,
    epoch: u16,
    sequence_number: u48,
    length: u16,
};

fn parseHeader(buf: []const u8) ?Header {
    if (buf.len < types.record_header_len) return null;

    // Version check
    if (buf[1] != 0xFE or buf[2] != 0xFD) return null;

    return .{
        .content_type = @enumFromInt(buf[0]),
        .epoch = std.mem.readInt(u16, buf[3..5], .big),
        .sequence_number = std.mem.readInt(u48, buf[5..11], .big),
        .length = std.mem.readInt(u16, buf[11..13], .big),
    };
}

fn writeHeader(
    out: []u8,
    content_type: types.ContentType,
    epoch: u16,
    sequence_number: u48,
    length: u16,
) void {
    out[0] = @intFromEnum(content_type);
    out[1] = 0xFE; // DTLS 1.2
    out[2] = 0xFD;
    std.mem.writeInt(u16, out[3..5], epoch, .big);
    std.mem.writeInt(u48, out[5..11], sequence_number, .big);
    std.mem.writeInt(u16, out[11..13], length, .big);
}

fn buildExplicitNonce(epoch: u16, seq: u48) [8]u8 {
    var nonce: [8]u8 = undefined;
    std.mem.writeInt(u16, nonce[0..2], epoch, .big);
    std.mem.writeInt(u48, nonce[2..8], seq, .big);
    return nonce;
}

fn buildNonce(implicit_iv: [4]u8, explicit_nonce: [8]u8) [12]u8 {
    var nonce: [12]u8 = undefined;
    @memcpy(nonce[0..4], &implicit_iv);
    @memcpy(nonce[4..12], &explicit_nonce);
    return nonce;
}

/// RFC 6347 §4.1.2.6 anti-replay sliding window.
/// Returns true if the sequence number is acceptable.
/// Updates the window on acceptance.
fn replayCheck(seq: u48, window: *u64, max_seq: *u48) bool {
    if (seq > max_seq.*) {
        const diff = seq - max_seq.*;
        if (diff >= 64) {
            // Gap too large — reset window entirely
            window.* = 1;
        } else {
            const shift: u6 = @intCast(diff);
            window.* = (window.* << shift) | 1;
        }
        max_seq.* = seq;
        return true;
    }

    const diff = max_seq.* - seq;
    if (diff >= 64) return false; // Too old

    const bit: u64 = @as(u64, 1) << @intCast(diff);
    if (window.* & bit != 0) return false; // Duplicate

    window.* |= bit;
    return true;
}

/// Undo a replay window update (used when decryption fails after replay check).
fn replayUndo(seq: u48, window: *u64, max_seq: *u48) void {
    if (seq == max_seq.*) {
        // Was the new max — can't easily undo the shift.
        // This is a known limitation; in practice, tampered records with
        // future sequence numbers could advance the window. Acceptable
        // since it only causes legitimate reordered packets to be dropped.
        return;
    }
    const diff = max_seq.* - seq;
    if (diff < 64) {
        const bit: u64 = @as(u64, 1) << @intCast(diff);
        window.* &= ~bit;
    }
}

```zig
const Prf = @import("Prf.zig");

pub const KeyBlock = struct {
    client_write_key: [16]u8,
    server_write_key: [16]u8,
    client_write_iv: [4]u8,
    server_write_iv: [4]u8,
};

/// Build PSK pre-master secret: <2-byte len><zeroes><2-byte len><psk>
pub fn buildPreMasterSecret(psk: []const u8, buf: []u8) []const u8 {
    const psk_len: u16 = @intCast(psk.len);
    const total = 2 + psk.len + 2 + psk.len;
    std.debug.assert(buf.len >= total);

    std.mem.writeInt(u16, buf[0..2], psk_len, .big);
    @memset(buf[2..][0..psk.len], 0);
    std.mem.writeInt(u16, buf[2 + psk.len ..][0..2], psk_len, .big);
    @memcpy(buf[4 + psk.len ..][0..psk.len], psk);
    return buf[0..total];
}

/// Derive master secret from pre-master secret.
pub fn deriveMasterSecret(
    pre_master: []const u8,
    client_random: [32]u8,
    server_random: [32]u8,
) [48]u8 {
    var seed: [64]u8 = undefined;
    @memcpy(seed[0..32], &client_random);
    @memcpy(seed[32..64], &server_random);

    var master: [48]u8 = undefined;
    Prf.prf(pre_master, "master secret", &seed, &master);
    return master;
}

/// Derive key block from master secret.
/// key_block = PRF(master, "key expansion", server_random || client_random)
pub fn deriveKeys(
    master_secret: [48]u8,
    server_random: [32]u8,
    client_random: [32]u8,
) KeyBlock {
    var seed: [64]u8 = undefined;
    @memcpy(seed[0..32], &server_random);
    @memcpy(seed[32..64], &client_random);

    // Need: client_write_key(16) + server_write_key(16) + client_write_iv(4) + server_write_iv(4) = 40
    var material: [40]u8 = undefined;
    Prf.prf(&master_secret, "key expansion", &seed, &material);

    return .{
        .client_write_key = material[0..16].*,
        .server_write_key = material[16..32].*,
        .client_write_iv = material[32..36].*,
        .server_write_iv = material[36..40].*,
    };
}
```

```zig
pub const HandshakeMessage = struct {
    msg_type: types.HandshakeType,
    message_seq: u16,
    body: []const u8,
};

/// Encode a DTLS handshake message (12-byte header + body).
/// Single fragment (no fragmentation for PSK messages).
pub fn encodeHandshakeMessage(
    msg_type: types.HandshakeType,
    message_seq: u16,
    body: []const u8,
    out: []u8,
) []const u8 {
    const total = 12 + body.len;
    std.debug.assert(out.len >= total);
    const body_len: u24 = @intCast(body.len);

    out[0] = @intFromEnum(msg_type);
    // length (3 bytes, big-endian)
    out[1] = @intCast((body_len >> 16) & 0xFF);
    out[2] = @intCast((body_len >> 8) & 0xFF);
    out[3] = @intCast(body_len & 0xFF);
    // message_seq
    std.mem.writeInt(u16, out[4..6], message_seq, .big);
    // fragment_offset = 0 (no fragmentation)
    out[6] = 0;
    out[7] = 0;
    out[8] = 0;
    // fragment_length = body length
    out[9] = out[1];
    out[10] = out[2];
    out[11] = out[3];

    @memcpy(out[12..][0..body.len], body);
    return out[0..total];
}

/// Decode a DTLS handshake message.
pub fn decodeHandshakeMessage(buf: []const u8) ?HandshakeMessage {
    if (buf.len < 12) return null;
    const body_len: u24 = (@as(u24, buf[1]) << 16) | (@as(u24, buf[2]) << 8) | buf[3];
    const frag_offset: u24 = (@as(u24, buf[6]) << 16) | (@as(u24, buf[7]) << 8) | buf[8];
    const frag_len: u24 = (@as(u24, buf[9]) << 16) | (@as(u24, buf[10]) << 8) | buf[11];

    // Only handle unfragmented messages (frag_offset=0, frag_len=body_len)
    if (frag_offset != 0 or frag_len != body_len) return null;
    if (buf.len < 12 + body_len) return null;

    return .{
        .msg_type = @enumFromInt(buf[0]),
        .message_seq = std.mem.readInt(u16, buf[4..6], .big),
        .body = buf[12..][0..body_len],
    };
}
```
